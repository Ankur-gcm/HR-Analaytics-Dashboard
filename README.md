# 📊 HR Analytics Dashboard

## 🔍 Project Overview

The **HR Analytics Dashboard** is an interactive Power BI dashboard designed to analyze employee attrition, demographics, salary trends, and job roles.
It helps HR teams make **data-driven decisions** to improve employee retention and workforce planning.

---

## 🚀 Key Metrics

* 👥 Total Employees: **961**
* 📉 Attrition Count: **133**
* 📊 Attrition Rate: **13.8%**
* 🎂 Average Age: **37**
* 💰 Average Salary: **6.3K**
* ⏳ Average Years at Company: **6.9**

## 📌 Key Insights

* Employees with **low salary (≤5K)** are more likely to leave
* **Age group 26–35** has the highest attrition
* **Laboratory Technicians** show maximum attrition
* Employees in **early tenure (0–2 years)** leave more frequently

## 🎯 Conclusion

This dashboard enables organizations to:

* Identify **high attrition segments**
* Improve **employee retention strategies**
* Optimize **salary and HR policies**
